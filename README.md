# To-Do-List

## Deskripsi
Aplikasi sederhana berbasis PHP untuk mencatat tugas harian.

## Fitur
- Tambah tugas
- Tandai tugas selesai
- Hapus tugas

## Struktur Folder
- 'index.php'-halamanan utama
- 'style.css'-file CSS

## Cara Menjalankan
1. Salin Folder ke 'htdocs/'
2. Jalankan XAMPP dan buka di 'http//localhost/To-Do-List'
